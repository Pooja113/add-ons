###  Cd Feedback Sample Form
This plugin contains a form for users to share their feedbacks and all users list is available for the administrator.
You can show form for front-end users by using [cdfeedbackform] shortcode and admin can also see the lists of all the users by [cdformdata] shortcode.

Shortcodes:
[cdfeedbackform] use this shortcode to display form. 
[cdfeedbackformdata] use this shortcode to display all enteries of this form in table format
